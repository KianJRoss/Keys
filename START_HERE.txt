KEYCHRON V1 COMMAND MACRO - BUILD & FLASH INSTRUCTIONS
================================================================================

Your firmware is READY TO COMPILE!

Files have been prepared and copied to the correct locations.

--------------------------------------------------------------------------------
STEP 1: COMPILE THE FIRMWARE
--------------------------------------------------------------------------------

EASIEST METHOD - QMK MSYS:

1. Press Windows key
2. Type "QMK MSYS" 
3. Open the QMK MSYS terminal (green icon)
4. Copy and paste this command:

   cd /c/CLIPALS/qmk_firmware && qmk compile -kb keychron/v1/ansi_encoder -km macro

5. Wait 30-60 seconds for compilation


ALTERNATIVE - Use the helper script:

   cd /c/CLIPALS/qmk_firmware && ./compile_macro.sh


Don't have QMK MSYS? Download: https://msys.qmk.fm/

--------------------------------------------------------------------------------
STEP 2: FLASH THE FIRMWARE
--------------------------------------------------------------------------------

After successful build:

1. Open QMK Toolbox
2. Click "Open" and select:
   C:\CLIPALS\qmk_firmware\.build\keychron_v1_ansi_encoder_macro.bin

3. Put keyboard in bootloader mode:
   - Unplug keyboard
   - Hold Space + B keys
   - Plug in USB while holding
   - Release after 2 seconds

4. Click "Flash" in QMK Toolbox

5. Done! Keyboard will reconnect automatically

--------------------------------------------------------------------------------
STEP 3: RUN THE AUTOHOTKEY SCRIPT
--------------------------------------------------------------------------------

Double-click: C:\CLIPALS\scripts\keychron_commands.ahk

You should see a notification: "Keychron Command Macro Handler Active"

--------------------------------------------------------------------------------
STEP 4: TEST IT!
--------------------------------------------------------------------------------

1. Rotate knob → Tooltip shows command name
2. Press knob → Command executes

Default commands:
  0: Open Terminal
  1: Launch VSCode  
  2: Run PowerShell script
  3: Open Calculator

--------------------------------------------------------------------------------
CUSTOMIZE COMMANDS (NO REFLASH NEEDED)
--------------------------------------------------------------------------------

Edit: C:\CLIPALS\scripts\keychron_commands.ahk

Change the "commands" array (around line 16)

Right-click AutoHotkey tray icon → Reload Script
